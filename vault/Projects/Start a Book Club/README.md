# Start a Book Club

**English:** Invite friends, choose a monthly book, and host the first discussion night. Purpose: shared reading and stronger bonds.

**Ελληνικά:** Προσκαλέστε φίλους, επιλέξτε ένα μηνιαίο βιβλίο και φιλοξενήστε την πρώτη βραδιά συζήτησης. Σκοπός: κοινή ανάγνωση και ισχυρότεροι δεσμοί.

**Deutsch:** Lade Freunde ein, wähle ein Monatsbuch und veranstalte den ersten Diskussionsabend. Ziel: gemeinsames Lesen und stärkere Bindungen.

**Sub‑folders**
- **Planning**: timelines, task lists, and brainstorming
- **Notes**: meeting notes, daily logs
- **Reference**: files, links, resources you might need
- **Archive**: move outdated material here as the project progresses
